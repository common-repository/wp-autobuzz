Allows you to automatically add full posts to Google Buzz

== Description ==

Allows you to add full or partial posts to Google Buzz automatically.  Allows one profile for the main feed and each author can add their own profile to their profile page.

== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.
== Frequently Asked Questions ==

Q: Can I do this myself without this plugin?

A: Yes, you can edit the template files to add the right links, but this method doesn't require that you know anything.

Q: I can't find the fields on author profile pages.

A: You must check the "Allow per author profile" box on the main plugin settings panel.
